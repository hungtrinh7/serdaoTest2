## Project Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/hungtrinh7/serdaoTest2.git
   cd serdaoTest2
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Make sure you have Android Studio opened and the emulator launched**

4. **Build and run the app**
   ```bash
   npm start
   ```
